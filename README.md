# React Todo App & ProfileCard Component

## ProfileCard Component — Plain English Summary
ProfileCard is a reusable React component that displays a person's profile information in a Bootstrap card. It accepts four props: name, bio, avatar, and location. The card shows a profile image at the top, followed by the person's name, a short bio, and their location. It includes a hover effect that lifts the card and adds a shadow when the mouse enters. It uses PropTypes to validate that all four props are required strings, and includes role="article" and aria-label for screen reader accessibility.

## Technologies Used
- React
- Bootstrap 5
- Vite
- PropTypes

## How to Run
1. Clone the repository
2. Run npm install
3. Run npm run dev
4. Open localhost:5173 in your browser
